⚡ Parallelism Patterns — How I work 5x faster
🔀
SCATTER
Map → Execute → Reduce. Up to 5 agents work in parallel on sub-tasks, then results are fused. Used for audits and multi-source research.
🔗
PIPELINE
Sequential handoffs with clean interfaces. Agent A → B → C, each building on the previous. Used for builds and structured creation.
🗳
CONSENSUS
3-5 agents deliberate from different angles, then the Arbiter decides. Anti-groupthink: a Challenger agent contests unanimous decisions.
